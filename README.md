# Komodo Palace

<img src="assets/komodopalace.png" alt="Komodo Palace Logo" width="150"/>

Website sederhana yang menampilkan informasi tentang komodo, habitatnya, dan fakta singkat lainnya.  
Dibuat untuk latihan web development menggunakan HTML, CSS, dan JavaScript.

## Fitur
- Ringan dan cepat.
- Responsif untuk mobile.

## Live Demo
https://komodo-palace.vercel.app

## Cara Menjalankan di Lokal
1. Clone repository:
   ```
   git clone https://github.com/alarixfr/komodo-palace.git
   ```
2. Masuk ke folder:
   ```
   cd komodo-palace
   ```
3. Buka `index.html` di browser.

## Deploy
Project ini di-deploy menggunakan **Vercel**.  
Push update ke GitHub lalu Vercel auto-build dan deploy.
